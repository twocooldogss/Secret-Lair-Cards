# 图片资源管理

## 目录结构
```
public/images/
├── drops/          # Secret Lair 掉落图片
├── cards/          # 卡牌图片
├── news/           # 新闻图片
├── investment/     # 投资分析图片
└── placeholders/   # 占位符图片
```

## 图片规格建议

### Drops 图片
- 尺寸：400x300px 或 800x600px
- 格式：JPG 或 PNG
- 命名：{slug}.jpg

### Cards 图片
- 尺寸：300x420px（标准卡牌比例）
- 格式：JPG 或 PNG
- 命名：{slug}.jpg

### News 图片
- 尺寸：400x250px
- 格式：JPG 或 PNG
- 命名：{slug}.jpg

## 获取来源

1. **官方资源**：Wizards of the Coast 官网
2. **免费图库**：Unsplash, Pexels, Pixabay
3. **MTG 社区**：Scryfall, Reddit
4. **AI 生成**：Midjourney, DALL-E

## 版权注意事项

- 确保图片有商用权限
- 避免使用受版权保护的图片
- 优先使用官方或开源资源
